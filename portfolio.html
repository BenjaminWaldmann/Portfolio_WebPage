Nutrient Tracker:

my Simple Nutrient Tracker was a small Project by me in python for CS50p as the Final Project i prepared
It uses the API by the USDA to get information on different products the user can search for.

It has a simple account management system, for different users on the same device,
a simle browser for different foods,
a menu where you can enter custom foods and products,
a cache file of eaten foods and used foods, so the api must not be called every time,
an overview of total consumed nutrients and
a feature to delete food, redo eating foods and deleting users, also wiping the cache.
